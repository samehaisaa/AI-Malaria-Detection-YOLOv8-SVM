# Tests

This directory contains test scripts for the AI Malaria Detection project.

## Test Files

- **test_image_processing.py**: Contains unit tests for the image processing utilities.
- **test_model.py**: Contains unit tests for the model utilities and functions.
- **test_app.py**: Contains unit tests for the main application and its components.

## Running Tests

To run the tests, use the following command:

```sh
pytest
```

Make sure you have all the necessary dependencies installed before running the tests. You can install the dependencies using the `requirements.txt` file located in the `app` directory:

```sh
pip install -r ../app/requirements.txt
```
